Flipper Multi Modal Dialogue System

Flipper Multi-Model Dialogue System (FlipperMMDS) is an addition to Flipper that makes it easier to develop a fully features Dialogue System. It is being developed by Siewart van Wingerden and Jelte van Waterschoot at the Humand Media Interaction group at the University of Twente.

Flipper is an information state (IS) dialogue manager (DM) originally developed by Mark ter Maat. Right now its documentation and latest version can be found at its github wiki. Since this is a layer on top of Flipper, reading its documentation is recommended (but this wiki should explain the basics). Since both are under active but early development the latest version of Flipper is required.

A good starting point is the github wiki.

This project is maintained by the Human Media Interaction Group (University of Twente)
